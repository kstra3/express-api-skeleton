Production-ready starter for building modern REST APIs

Get Started View API

Features

Security First

Helmet, CORS, rate limiting, input sanitization (XSS, NoSQL injection, HPP), and JWT auth out of the box.

Production Ready

Graceful shutdown, health checks, request timeouts, logging, and Docker support with multi-stage builds.

Developer Friendly

ESLint + Prettier, Jest tests, Swagger docs, hot reload via nodemon, and Makefile for common tasks.

Extensible

Modular architecture: drop new feature folders under src/modules/ with routes, controllers, and validations.

Quick Start

git clone <repo-url>
cd kilo-test
cp .env.example .env
npm install
npm run dev

Server starts on http://localhost:3000. Swagger UI at /api-docs.

API Endpoints

Method Path Auth Description
GET/api/healthNoHealth check
GET/api/health/readyNoReadiness with dep checks
GET/api/usersNoList users
POST/api/usersNoCreate user
POST/api/auth/registerNoRegister (demo JWT)
POST/api/auth/loginNoLogin (demo JWT)
POST/api/uploadNoUpload file (multipart)
Note: Auth and upload are demo implementations. Add real DB-backed user service and file storage for production.
